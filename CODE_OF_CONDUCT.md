# Code Of Conduct

Mangou welcomes constructive technical discussion and contribution.

When participating in this project, follow these rules:

- Be respectful and focus on facts and technical issues
- Do not engage in personal attacks, harassment, discrimination, or deliberate provocation
- Do not publish private information, secrets, API keys, or sensitive data
- For disputed topics, provide reproducible evidence and concrete context first

Maintainers may remove inappropriate content and restrict repeated behavior that damages collaboration.

For maintainer contact, email `mangou@mangou.art`.
