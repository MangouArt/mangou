# Security Policy

If you discover a security issue in Mangou, do not open a public issue.

Report it by email:

- `mangou@mangou.art`

If the issue is related to commercial deployment, licensing boundaries, or partnership concerns, you may also copy:

- `business@mangou.art`

## What To Include

- Issue type
- Affected scope
- Reproduction steps
- Possible exploitation path
- Suggested mitigation, if available

We will review the report and follow up as quickly as possible.
